# Need Gone MVP

## Version 1.0 Goal

A person should be able to give away an item without needing awkward back-and-forth messaging.

## MVP Must Have

### Accounts

- [x] Google sign in
- [x] Sign out
- [ ] Basic user profile
- [ ] Reputation summary

### Listings

- [x] Create listing
- [x] Upload photo
- [x] Browse listings
- [x] View listing details
- [x] My Listings page
- [x] Mark item gone
- [ ] Show pickup information properly
- [ ] Listing expiration
- [ ] Search/filter listings

### Pickup Setup

- [x] Porch pickup option
- [x] Public meetup option
- [x] Public location
- [x] Private address
- [x] Pickup window
- [x] Pickup instructions
- [x] Meetup location
- [x] Meetup time options

### Pickup Requests

- [ ] Claimer can request pickup
- [ ] Claimer can choose pickup time
- [ ] Giver can view requests
- [ ] Giver can approve one request
- [ ] Giver can decline requests
- [ ] Approved claimer can see pickup details
- [ ] Item automatically becomes pending after approval
- [ ] Giver can mark completed
- [ ] Giver can mark missed pickup
- [ ] System can offer item to next requester

### Reputation

- [ ] Track completed pickups
- [ ] Track missed pickups
- [ ] Track canceled pickups
- [ ] Track items given
- [ ] Track items claimed

### Launch

- [ ] Polish navigation
- [ ] Clean up item cards
- [ ] Mobile responsiveness
- [ ] Deploy to Vercel
- [ ] Invite 10 test users

## Not MVP Yet

These are good ideas, but not required for version 1.0.

- In-app chat
- Payments
- Premium profiles
- Profile skins
- AI features
- Full notification system
- Advanced maps
- Mobile app
- Business accounts

## Possible Version 1.5

- Giving Circles
- Invite friends
- Circle-only listings
- Friends-first listings
- Public release after 24 hours
- Group reputation
- Circle notifications
